CounselVault Git + Vercel Ready

Files included:
- index.html
- vercel.json

How to deploy:
1. Create a new GitHub repo
2. Upload these files to the root of the repo
3. Import the repo into Vercel
4. Deploy

Important:
- This version is online-ready and easy to deploy
- It is NOT a shared multi-user database app yet
- Data is saved in each person's browser using localStorage

For true remote firm use, the next build is:
- shared database
- real staff auth
- server-side storage
- live syncing across users
